A headless WordPress starter theme optimized for GraphQL use.

== Description ==

GraphQL Starter is a specialized WordPress theme designed to function purely as a headless GraphQL API endpoint. It strips away all frontend rendering and traditional WordPress theme features, focusing solely on providing a clean, efficient GraphQL API interface.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
